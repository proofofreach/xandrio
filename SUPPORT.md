# Support

- **Bug:** open a bug report with version, deployment mode, steps, expected result, actual result, and redacted logs.
- **Feature or documentation request:** open the matching issue template.
- **Provider outage or compatibility problem:** use the provider report template. Providers are independent services; an outage is not necessarily a Xandrio defect.
- **Security issue:** use the private path in [SECURITY.md](SECURITY.md), not a public issue.
- **Legal, copyright, or jurisdiction-specific question:** seek qualified local advice. Maintainers cannot decide whether a provider, work, or use is permitted for an operator.

Never include passwords, API keys, session tokens, book files, voice references, or generated audio in a public report.
